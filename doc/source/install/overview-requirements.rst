
Installation requirements
=========================

Salt master
------------

Required items:

-  Ubuntu 14.04 LTS (Trusty Tahr)
-  Synchronized network time (NTP) client.
-  Python 2.7 or later.

Salt minions
------------

Required items:

-  Ubuntu Server 14.04 LTS (Trusty Tahr) 64-bit operating system
-  Synchronized NTP client.
